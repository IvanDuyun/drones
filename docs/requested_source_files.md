# Files Needed From The Previous Repository

Для следующего шага нужны не все файлы подряд, а только те, которые закрывают интеграцию.

## Обязательно

1. Файл, на основе которого вы обучали `PPO`:
   - ваш измененный `learn.py`, если вы его уже адаптировали;
   - либо исходный `gym_pybullet_drones/examples/learn.py`, если правок еще нет.

2. Файл или пример, где запускается/оценивается обученная модель:
   - любой ваш локальный скрипт оценки;
   - если своего еще нет, тогда пример из библиотеки, где модель загружается и прогоняется.

3. Файл среды, которую вы реально хотите использовать для hover-задачи:
   - конкретная env-реализация из `gym-pybullet-drones`;
   - либо ссылка на точный путь в библиотеке, если файл менять не нужно.

## Желательно

4. Пример `PID`-контроллера или скрипт, который вы хотите использовать в сравнении `PID vs PPO`.

5. Любые ваши локальные правки для `Colab`, если они уже есть.

6. Список версий, на которых у вас что-то уже запускалось:
   - Python;
   - `gym-pybullet-drones`;
   - `stable-baselines3`;
   - `pybullet`.
